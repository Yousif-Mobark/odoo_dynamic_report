üöÄ Getting Started

1. Installation

# Install Python dependencies pip install python-docx lxml Pillow # Install module in Odoo Apps ‚Üí Update Apps List ‚Üí Search "Dynamic Report Builder" ‚Üí Install

2. Create Your First Template

  1. Navigate to Report Builder ‚Üí Templates
  2. Click Create
  3. Enter a name and select an Odoo model (e.g., Contact, Sales Order)
  4. Upload a DOCX file with field placeholders
  5. Click Preview to test with real data

3. Template Syntax

{{name}} {{partner_id.name}} {{date_order|date:'%Y-%m-%d'}} {{amount_total|number:'0.2f'}} {{#order_line}} {{product_id.name}} | {{quantity}} | {{price_unit}} {{/order_line}}

üîß Advanced Features

Custom Formatters

Apply formatting to field values:

  • date:'format' - Format dates (e.g., %Y-%m-%d)
  • number:'format' - Format numbers (e.g., 0.2f for 2 decimals)
  • upper - Convert to uppercase
  • lower - Convert to lowercase
  • title - Title case
